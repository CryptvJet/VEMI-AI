About VEMi AI Chat

VEMi AI Chat is an AI-powered chatbot designed for intelligent and interactive conversations. It supports **text-based communication, voice-to-text input, session tracking, and database management**.

üîß Features & Functionality

  • AI Chat System: Accepts user input and provides intelligent responses.
  • Training System: Unanswered questions are stored for training.
  • Admin Panel: Allows managing responses, deleting unanswered questions, and viewing chat history.
  • Session Logging: Stores user interactions and IP addresses for chat history tracking.
  • Voice-to-Text: Users can send messages using speech recognition.
  • Pagination: Trained responses and session logs support pagination for better organization.

üìå How VEMi AI Chat Works

  1. User sends a message via the chat interface.
  2. The message is processed by ai-chat.php.
  3. If the chatbot knows the response, it returns the answer.
  4. If the chatbot **doesn't** know the answer, the message is logged as "unanswered" for training.
  5. Admins can manage responses via admin.php.
  6. Chat history is stored in the **database** and can be viewed in the **Session Logs**.

üìå How to Set Up the Database

1Ô∏è‚É£ Create the Database

Run the following SQL command to create the database:

CREATE DATABASE vemite5_ai;

2Ô∏è‚É£ Set Up Tables

Execute the following SQL to create necessary tables:


CREATE TABLE responses (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_message TEXT NOT NULL,
    bot_response TEXT NOT NULL,
    response_type ENUM('Master', 'AI', 'UP') DEFAULT 'AI',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE session_logs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    session_id VARCHAR(255) NOT NULL,
    ip_address VARCHAR(45),
    user_message TEXT NOT NULL,
    bot_response TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

3Ô∏è‚É£ Configure Database in PHP

Update the `config.php` file with your database credentials:


$servername = "localhost";
$username = "your_username";
$password = "your_password";
$database = "vemite5_ai";
$conn = new mysqli($servername, $username, $password, $database);
if ($conn->connect_error) { die("Database connection failed"); }

üì° How to Deploy VEMi AI Chat

üîπ Step 1: Upload Files to Server
