Project Scope

VEMi AI is structured into multiple components, ensuring a **scalable, efficient, and easy-to-maintain** chatbot experience. The core components include:

  1. Frontend: Chat UI for users to interact with the bot (HTML, CSS, JS).
  2. Backend: PHP scripts for processing chat requests and managing responses.
  3. Database: MySQL-based storage for chat logs, responses, and user sessions.
  4. Admin Panel: A control interface for training AI responses and managing logs.

  2. ‚úÖ Create a **new database** named `vemite5_ai`.
  3. ‚úÖ Import the following SQL structure:

    id INT AUTO_INCREMENT PRIMARY KEY,
    user_message TEXT NOT NULL,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
